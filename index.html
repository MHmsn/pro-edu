Frequently Asked Questions

Here you will get answers to some of the most frequently asked questions

Flexbox is a one dimensional layout system, Where Grid is a two dimensional layout system with rows and columns Usually flex is used when we have to edit an one dimensional layout, or not more than one row. with flexbox we can justify contents or align items. but when we have two dimensional layout, or more than one row, we can use grid to finish the layout much faster.

Bootstrap is the most popular CSS framework and Tailwind is an utility first CSS framework. We use bootstrap for quickly building up a responsive website. but bootstrap lacks some specific customization features. For very specific customization we have tailwind. As it is an utility first CSS framework, we can use tailwind's highly customizable features to quickly setup custom user interfaces.

CSS box model is an imagined box that is wrapped around every HTML element. It consists of main content, padding, border, and margin. CSS box model is used when we modify an element's design and layout using the 4 aforementioned properties.

Semantic Tags are meaningful tags that provide meaning instead of just presenting. Using semantic tags helps to make HTML file more understandable and simplifies it by differentiating different sections and layout of an webpage.
